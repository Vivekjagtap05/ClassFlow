A Java Swing-based School Management System designed to manage students, attendance, marks, fees, activities, and academic records through a user-friendly desktop interface.

---

## 🚀 Features

### 👨‍🎓 Student Management

* Add new students
* Update student details
* Delete student records
* View all students

### 📅 Attendance Management

* Mark attendance
* Track attendance history
* View attendance records

### 📊 Marks Management

* Add student marks
* Update marks
* Monitor academic performance

### 💰 Fee Management

* Record fee payments
* Track fee status
* Manage student fee details

### 🏆 Activity Management

* Manage extracurricular activities
* Track student participation

### 📚 Academic Dashboard

* View academic information
* Monitor student progress

### 🔐 Authentication

* Secure login system
* Admin access management

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/ClassFlow.git
```

### Open Project

Import the project into Eclipse IDE.

### Configure Database

Create a MySQL database:

```sql
CREATE DATABASE classflow;
```

Update database credentials in your JDBC configuration:

```java
String url = "jdbc:mysql://localhost:3306/classflow";
String username = "root";
String password = "your_password";
```

### Run Project

Run the main application file:

```bash
ClassFlowApp.java
```
